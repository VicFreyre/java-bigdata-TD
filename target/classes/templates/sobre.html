Sobre o Sistema

Sistema de Matrículas

Sistema completo de gerenciamento de matrículas desenvolvido com Spring Boot.

Tecnologias Utilizadas
  • Java 17.0.12
  • Spring Boot 3.2.0
  • Spring Data JPA
  • PostgreSQL 17
  • Thymeleaf
  • Bootstrap 5.3.0
  • Jakarta Validation
  • Lombok
  • ModelMapper
Funcionalidades
  • CRUD de Alunos: Cadastro, edição, exclusão e listagem de alunos com busca
  • CRUD de Cursos: Gerenciamento completo de cursos com categorias
  • Matrículas: Criação, cancelamento e visualização de matrículas
  • Relatórios:
    • Alunos por Curso
    • Cursos Mais Procurados
    • Matrículas por Período
Arquitetura

O sistema foi desenvolvido seguindo os princípios de POO rigorosa:

  • Entidades com encapsulamento completo
  • DTOs para entrada e saída de dados
  • Services com interface e implementação
  • Controllers sem lógica de negócio
  • Repositórios JPA para acesso a dados
